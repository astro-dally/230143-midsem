# Dataset Description

The dataset used in this experiment is generated programmatically using `sklearn.datasets.make_classification`.

Dataset parameters:
- 500 samples
- 5 features
- 3 informative features
- class_sep = 1.0
- flip_y = 0.05

This synthetic dataset is used to simulate a binary classification task suitable for evaluating kernel-based SVM classifiers.

No external dataset download is required. The dataset is generated directly within the notebook `task_2_1.ipynb`.
